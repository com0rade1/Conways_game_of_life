# Conways_game_of_life
 

Изначально требуется также задать размер игрового поля (количество клеток, которым оно будет замощено). После того, как нужное положение ползунка выбрано, нужно нажать кнопку "задать масштаб" для формирования массива требуемого размера. После того, как клетки отрисуются на игровом поле, можно либо сгенерировать массив клеток произвольным образом, либо заполнить поле с помощью нажатий левой кнопкой мыши по нужной клетке. Кроме того, автоматически можно создать два состояния - "Вирус" - создаются сады Эдема, достаточно лишь заполнить клетку, смежную со сторонами двух чёрных квадратов 2x2 и постоянство системы будет нарушено. Крест добавлен из эстетических соображений. Игра реализована на торе, то есть левая граница поля совмещена с правой, а верхняя - с нижней.
